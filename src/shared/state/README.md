# State — глобальное состояние игры

Мутабельный синглтон `state` с полным состоянием боя + генераторы имён юнитов.

## API

```ts
import {
  createInitialState,
  nextAllyName,
  nextEnemyName,
  resetNameCounters,
  state,
} from '@/shared/state';
```

| Экспорт                | Описание                                       |
| ---------------------- | ---------------------------------------------- |
| `state`                | Текущее состояние игры (мутабельный объект)    |
| `createInitialState()` | Фабрика начального состояния                   |
| `resetNameCounters()`  | Сбросить счётчики имён (при старте нового боя) |
| `nextAllyName()`       | Следующее имя союзника (`Сокол-1`, `Орёл-1`…)  |
| `nextEnemyName()`      | Следующее имя врага (`Фантом-1`, `Тень-1`…)    |

## Структура

- `index.ts` — состояние, генераторы имён
- Типы состояния (`GameState`) определены в `@/shared/types`
