# Input — система ввода

Преобразует нажатия клавиш и движение мыши в игровые действия.

## API

```ts
import { actions, aim, getKeyMap, inputSystem, setKeyMap } from '@/shared/input';
```

| Экспорт       | Описание                                             |
| ------------- | ---------------------------------------------------- |
| `actions`     | Объект `Record<Action, boolean>` — что нажато        |
| `aim`         | `{ x, y }` — нормализованная позиция прицела         |
| `setKeyMap()` | Установить новую раскладку клавиш                    |
| `getKeyMap()` | Получить текущую раскладку (readonly)                |
| `inputSystem` | `GameSystem` — обновляет `actions`/`aim` каждый кадр |

## Действия (Action)

| Действие    | Описание         |
| ----------- | ---------------- |
| `thrust`    | Ускорение вперёд |
| `brake`     | Торможение       |
| `boost`     | Форсаж           |
| `fire`      | Стрельба         |
| `rollLeft`  | Крен влево       |
| `rollRight` | Крен вправо      |

## Структура

- `types.ts` — тип `Action`
- `index.ts` — состояние ввода, маппинг клавиш, `GameSystem`
